# Frontend Tests

Tests for the LLM Council frontend application.

## Running Tests

### Install dependencies:
```bash
cd frontend
npm install
```

### Run tests:
```bash
npm test
```

### Run with UI:
```bash
npm run test:ui
```

### Run with coverage:
```bash
npm run test:coverage
```

## Test Files

- `api.test.js` - API client tests
- `setup.js` - Test configuration

## Framework

Using Vitest with React Testing Library.
